How to Play

The Game

Players target numbers 1 through 7 in order. Each player throws 3 darts per round, scoring only on that round's target number. Highest total score after 7 rounds wins.

Scoring

  • Single = the target number's value
  • Double = 2× the value
  • Triple = 3× the value
  • Miss = 0

Shanghai!

If you hit a Single, Double, AND Triple of the target number all in one turn — you win instantly, regardless of score!
